````
。/antario 仿安踏奥运移动端效果
使用原生javascript进行开发;
开发中用到的js文件: 
	1. 运动函数->move.js
	2. 工具函数->myTools
	3. 主要逻辑调用->index.js
	4. 页面中用的数据->data.js
web优化方面:
	1. 减少资源加载次数,图片压缩
项目描述: 
  1.网站由大量图片使用css3特性形成立体效果，需要对所有img进行预加载
  2.图片加载完成，进场动画，主要是使用css3的translate3d控制元素的旋转，缩放，远近实现效果
  3.进场动画过后，碎片炸裂，围绕logo旋转效果，主要js控制初始位置，计算轨迹，实现效果。
  4.主体部分，可以拖动旋转，可以重力感应
 
````
